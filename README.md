# Análisis de Tiendas - Alura Store

Este proyecto analiza el desempeño de cuatro tiendas de la cadena Alura Store utilizando Python en Google Colab.

## Objetivo

Determinar qué tienda debería vender el Sr. Juan basándose en diferentes indicadores de desempeño.

## Herramientas utilizadas

- Python
- Pandas
- Matplotlib
- Google Colab

## Análisis realizados

Se analizaron los siguientes aspectos:

- Ingresos totales por tienda
- Ventas por categoría de productos
- Calificación promedio de los clientes
- Productos más y menos vendidos
- Costo promedio de envío

## Conclusión

Tras analizar los datos se determinó que **la Tienda 4 presenta el menor desempeño en términos de ingresos**, por lo que se recomienda vender esta tienda para invertir en un nuevo emprendimiento.
